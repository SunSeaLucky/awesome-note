## 设置用户名和邮箱

全局

```bash
git config --global user.name SunSeaLucky
git config --global user.email sunsealucky@gmail.com
```

当前工作区

```bash
git config user.name SunSeaLucky
git config user.email sunsealucky@gmail.com
```
